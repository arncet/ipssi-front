# IPSSI-site-front

##Installation :

  Clone the project
  `git clone git@github.com:Dev6-CKS/IPSSI-site-front.git`

  Open the folder
  `cd IPSSI-site-front`

  Install node modules:
  `npm install`

  Run project:
  `npm start`

  Open the following link in your browser:
  `http://localhost:3000`
